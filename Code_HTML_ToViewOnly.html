Dataset Background :¶

About Dataset:¶

This dataset contains around 50k+ traffic-related violation records.

Target:¶

Dependent values: is_arrested

Attribute Information:¶

  • stop_date:
    • Description: Date of violation
    • Data type: Interval (Date)
    • Range: 1/1/2006 - 9/9/2011
    • Format: MM/DD/YYYY
    • Scale: None
    • Missing values: No
    • Error: No
  • stop_time:
    • Description: Time of violation
    • Data type: Interval (Time)
    • Range: 1:55 - 5:23
    • Format: 00:00 (mins)
    • Scale: None
    • Missing values: Yes - 1 missing
    • Error: No
  • driver_gender:
    • Description: Gender of violators
    • Data type: Dichotomous
    • Range: M - F
    • Format: MM/DD/YYYY
    • Scale: None
    • Missing values: Yes - 3386
    • Error: No
  • driver_age:
    • Description: Age of violators
    • Data type: Interval
    • Range: 20 - 84
    • Format: 00
    • Scale: None
    • Missing values: Yes - 3620 missing
    • Error: No
  • driver_race:
    • Description: Race of violators
    • Data type: Nominal
    • Range: { 'Asian', 'Black', 'Hispanic', 'Other', 'White' }
    • Format: None
    • Scale: None
    • Missing values: Yes - 3385 missing
    • Error: No
  • violation:
    • Description: Violators
    • Data type: Nominal
    • Range: {'Equipment', 'Moving violation', 'Other' 'Registration/plates', 'Speeding'}
    • Format: None
    • Scale: None
    • Missing values: Yes - 3385 missing
    • Error: No
  • search_conducted:
    • Description: Whether the criminal search is conducted on the violators or not
    • Data type: Dichotomous
    • Range: True - False
    • Format: None
    • Scale: None
    • Missing values: Yes - 1 missing
    • Error: No
  • stop_outcome:
    • Description: The result of the violation process
    • Data type: Nominal
    • Range: {'Citation', 'Arrest Driver','N/D', 'Warning', 'Arrest Passenger', 'No Action'}
    • Format: None
    • Scale: None
    • Missing values: Yes - 3385 missing
    • Error: No
  • is_arrested:
    • Description: Whether the violators is arrested or not
    • Data type: Dichotomous
    • Range: True - False
    • Format: None
    • Scale: None
    • Missing values: Yes - 3385 missing
    • Error: No
  • stop_duration:
    • Description: The violators stop duration
    • Data type: Nominal ( Discretization )
    • Range: {'0-15 Min', '16-30 Min' , '30+ Min'}
    • Format: None
    • Scale: None
    • Missing values: Yes - 3385 missing
    • Error: No
  • drugs related stop:
    • Description: Whether a person was involved in drugs crime
    • Data type: Dichotomous
    • Range: True - False
    • Format: None
    • Scale: None
    • Missing values: Yes - 1 missing
    • Error: No

    </ul>

# how many Arrested vs Not Arrested

def visualize_target(df):
    fig, ax = plt.subplots()
    fig.canvas.draw()
    df.is_arrested.value_counts().plot(kind="bar", width=0.2, figsize=(8, 4), color = ["royalblue","darkorange"])
    plt.ylabel("Count", fontsize=12)
    plt.title("Arrested vs. Not Arrested before search conduct")
    for i in ax.patches:
        percentage = '{:.1f}%'.format(100 * i.get_height()/len(df.is_arrested))
        x = i.get_x() + i.get_width() - 0.15
        y = i.get_height()
        ax.annotate(percentage, (x, y))
    ax.set_xticklabels(["Not Arrested", "Arrested"], rotation = 0);
    
visualize_target(traffic_df)

def SVM_hyperparameterOptimization( X_train_std, y_train,X_test, y_test, text='over Sampling' ):
    
    """

    X_train_std: X train dataset Standardized
    y_train: y train dataset (target)
    X_test: X test dataset 
    y_test: y test dataset (target)
    text: imbalanced data technique  
    
    """
    
    # kernal: Specifies the kernel type to be used in the algorithm. 
    kernels = ['linear',  'sigmoid']
    
    #C: Regularization parameter. The strength of the regularization is inversely proportional 
    #to C. Must be strictly positive. The penalty is a squared l2 penalty.
    C = [1.0, 1.5]
    
    
    # save the acc result 
    acc = []
    
    
    # for each kernal, loop over the hyperparameter and fit the model 
    for kernel in kernels:
        
        acc_kernel  = []
        
        for i in range(len(C)):

            # init the model with C and epsilon
            svm_model= svm.SVC(kernel=kernel, C=C[i], probability=True)

            # fit SVM
            svm_model.fit(X_train_std, y_train)
        

            # predict 
            predict_svm = svm_model.predict(X_test)

           # test the model accuracy 
            mae_svm = mean_absolute_error(y_test, predict_svm)
            accuracy = accuracy_score(y_test, predict_svm)

          
            # print: 
            print(' SVM with kernel {0} and {1} technique, C= {2:0.3f}  MSE: {3:0.3f}'.format(kernel, text, C[i],  mae_svm))
            print(' SVM with kernel {0} and {1} technique, C= {2:0.3f}  Acc: {3:0.3f}'.format(kernel, text, C[i],  accuracy))
            
            
            precision = precision_score(y_test, predict_svm)
            print('SVM with kernel {0} and {1} technique, C= {2:0.3f} PPTV: {3:0.3f}'.format(kernel, text, C[i], precision))
            
            TP, FP, TN, FN = confusion_matrix(y_test, predict_svm).ravel()
            print('TP: {0:0.3f} FP: {1:0.3f} TN: {2:0.3f}  FN: {3:0.3f}'.format(TP, FP, TN, FN))
            
            if (TN+FN) == 0:
                FN= 1
            NPV = TN/(TN+FN)
            print('SVM with kernel {0} and {1} technique, C= {2:0.3f} NPV: {3:0.3f}'.format(kernel, text, C[i], NPV))
            
            recall = TP/(TP+FN)
            print('SVM with kernel {0} and {1} technique, C= {2:0.3f} Recall: {3:0.3f}'.format(kernel, text, C[i], recall))
            
            specificity = TN / (TN+FP)
            print('SVM with kernel {0} and {1} technique, C= {2:0.3f} Specificity: {3:0.3f}'.format(kernel, text, C[i], specificity))
            
          
            F1 = 2* ((precision*recall)/(precision+recall))
            print('SVM with kernel {0} and {1} technique, C= {2:0.3f} F1: {3:0.3f}'.format(kernel, text, C[i], F1))


            
            print('-----------------------------------------------------------------------------------------------------------------------------------')
            
            acc_kernel.append(mae_svm)
            
        # append the acc for visualize
        acc.append(acc_kernel)

    df = pd.DataFrame(acc,
                     index=kernels,
                     columns=pd.Index([ 'MSE with C=1.0', 'MSE with C=1.5'], 
                     name='Kernels'))

    print(df)
    df.plot(kind='bar',figsize=(20,8))

    ax = plt.gca()
    pos = []

    for bar in ax.patches:
        pos.append(bar.get_x()+bar.get_width()/2.)
        ax.annotate(str(bar.get_height()), (bar.get_x() * 1.005, bar.get_height() * 1.005))



    ax.set_xticks(pos,minor=True)
    lab = []
    for i in range(len(pos)):
        l = df.values[i//len(df.index.values)][0]
        lab.append(l)



    plt.title("SVM Hyperparameter Optimization")
    plt.show()

    X_test: X test dataset

    """

        LogisticRegression_model = LogisticRegression(solver=solver, random_state=42)

        LogisticRegression_model.fit(X_train_std, y_train)
